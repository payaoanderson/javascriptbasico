Desvio Condicional "if" em JavaScript: Tomando Decisões no Código

O desvio condicional "if" é uma estrutura fundamental em JavaScript e em muitas outras linguagens de programação. Ele permite que você tome decisões no seu código, executando blocos de código diferentes com base em condições específicas. Neste tópico, vamos explorar o uso do "if" em JavaScript.

Principais Pontos:

  1. Sintaxe Básica:

    2. A estrutura "if" é usada para avaliar uma condição. Se a condição for verdadeira, o bloco de código dentro do "if" é executado.

    if (condição) {
    // Código a ser executado se a condição for verdadeira
}
  2. Condições:

    3. A "condição" é uma expressão que avalia para verdadeira (true) ou falsa (false). Se a condição for verdadeira, o código dentro do "if" será executado; caso contrário, ele será ignorado.

    Criar: 01-desvio-condicional-se.html
                
var idade = 18;
if (idade >= 18) {
    document.write("Pode votar!");
}
  3. "else" (Senão):

    4. Você pode adicionar uma cláusula "else" após um bloco "if" para especificar um código a ser executado quando a condição for falsa.

    Criar: 02-desvio-condicional-se-senao.html
    
var idade = 16;
if (idade >= 18) {
    document.write("Pode votar!");
} else {
    document.write("Não pode votar.");
}
  4. "else if" (Senão Se):

    5. Quando você tem várias condições a serem testadas, você pode usar "else if" para verificar condições adicionais após o "if" inicial.

    Criar: 03-desvio-condicional-se-senaose-senao.html
    
var pontuação = 75;
if (pontuação >= 90) {
    document.write("Aprovado com distinção!");
} else if (pontuação >= 60) {
    document.write("Aprovado");
} else {
    document.write("Reprovado");
}
  5. Operadores Condicionais:

    6. Operadores condicionais, como "==", "!=", ">", "<", ">=" e "<=", são usados para comparar valores e criar condições em declarações "if".

    Criar: 04-desvio-condicional-operadores-condicionais.html
    
var x = 10;
var y = 5;
if (x > y) {
    document.write("x é maior que y.");
}
  6. Aninhamento de "if":

    7. Você pode aninhar várias instruções "if" dentro de outras instruções "if" para lidar com lógica complexa.

    Criar: 05-desvio-condicional-se-aninhado.html
                
var x = 10;
var y = 5;
if (x > 5) {
    if (y > 2) {
        document.write("x é maior que 5 e y é maior que 2.");
    }
}
  7. Operadores Lógicos:

    8. Você pode usar operadores lógicos, como "&&" (E) e "||" (OU), para combinar condições em declarações "if" mais complexas.

    Criar: 06-desvio-condicional-operadores-logicos.html
                
var aprovado = true;
var frequencia = 80;
if (aprovado && frequencia >= 75) {
    document.write("Aprovado por nota e frequência.");
}
