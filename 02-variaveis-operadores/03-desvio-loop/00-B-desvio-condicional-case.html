Estrutura de Controle "switch" em JavaScript

A estrutura de controle "switch" é usada em JavaScript para realizar testes de múltiplos casos (cases) com base no valor de uma expressão. Ela é especialmente útil quando você precisa tomar decisões com base em várias opções possíveis. O "switch" é frequentemente utilizado para criar declarações de caso (case statements).

Sintaxe Básica do "switch":

    
switch (expressão) {
    case valor1:
        // Código a ser executado se expressão igual a valor1
        break;
    case valor2:
        // Código a ser executado se expressão igual a valor2
        break;
    // Outros casos
    default:
        // Código a ser executado se nenhum caso corresponder
}

No exemplo acima, a expressão é avaliada e o "switch" direciona o fluxo do programa para o caso correspondente. Cada caso contém um bloco de código a ser executado. A instrução "break" é usada para sair do "switch" após a execução do caso correspondente.

Exemplo de Uso:

    
Criar: 07-controle-switch.html
                
var diaDaSemana = 3;

switch (diaDaSemana) {
    case 1:
        document.write("Hoje é segunda-feira.");
        break;
    case 2:
        document.write("Hoje é terça-feira.");
        break;
    case 3:
        document.write("Hoje é quarta-feira.");
        break;
    case 4:
        document.write("Hoje é quinta-feira.");
        break;
    case 5:
        document.write("Hoje é sexta-feira.");
        break;
    case 6:
        document.write("Hoje é sábado.");
        break;
    case 7:
        document.write("Hoje é domingo.");
        break;
    default:
        document.write("Dia inválido.");
}

Neste exemplo, o "switch" é usado para determinar o dia da semana com base no valor da variável "diaDaSemana". Cada caso corresponde a um dia da semana, e o bloco de código apropriado é executado.
