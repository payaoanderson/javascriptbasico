Laço de Repetição "for" para Cálculo da Tabuada

O laço de repetição "for" é uma estrutura de controle essencial em JavaScript para a realização de tarefas repetitivas. Neste exemplo, usaremos um laço "for" para calcular a tabuada de um número escolhido e exibi-la diretamente na página HTML.

Exemplo de Uso:

    
Criar: 09-laco-de-repeticao-for.html<br>;                                            
for (var i = 0; i < 5; i++) {
    document.write("Número: " + i + "<br>");
}

Neste exemplo, o "for" é usado para iterar de 0 a 4. A variável de controle "i" é inicializada como 0, e o laço continua enquanto "i" for menor que 5. Após cada iteração, "i" é incrementada em 1.

O resultado será:

Número: 0
Número: 1
Número: 2
Número: 3
Número: 4

O laço "for" é uma ferramenta poderosa para executar tarefas repetitivas e percorrer coleções de dados, como arrays. Ele permite automatizar processos e simplificar o código, tornando-o mais eficiente e legível.

    
Criar: 10-laco-de-repeticao-for-tabuada.html<br>                                    
    var numero = 7; // Substitua pelo número desejado
    var resultado = "";

    for (var i = 1; i <= 10; i++) {
        resultado += numero + " x " + i + " = " + (numero * i) + "<br>";
    }

    document.write(resultado);
