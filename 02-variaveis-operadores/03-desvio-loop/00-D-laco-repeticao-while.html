Laço de Repetição "while" em JavaScript

O laço de repetição "while" é uma estrutura de controle em JavaScript que permite executar um bloco de código enquanto uma condição especificada for verdadeira. É útil quando você não sabe antecipadamente quantas vezes o código deve ser executado, mas deseja continuar até que uma condição seja atendida.

Sintaxe Básica do "while":

    
while (condição) {
    // Código a ser repetido
}

No exemplo acima, o bloco de código dentro do "while" é executado repetidamente enquanto a condição especificada for verdadeira. A condição é avaliada antes de cada iteração, e se a condição se tornar falsa, o laço é interrompido.

Exemplo de Uso:

Aqui está um exemplo de uso do "while" para contar de 1 a 5:

    
Criar: 11-laco-de-repeticao-while.html

var numeroInicial = 1;
var numeroFinal = 5
while (numeroInicial <= numeroFinal) {
    document.write("Contagem: " + numeroInicial + "<br>");
    numeroInicial++;
}

Neste exemplo, o "while" é usado para repetir a contagem enquanto o valor do contador for menor ou igual a 5. O contador é incrementado em cada iteração.

    var nomes = [];
    var nome;
    var contador = 0;
    
    while (true) {
        nome = prompt("Digite um nome ou 'sair' para encerrar:");
        if (nome === "sair") {
            break;
        }
        nomes.push(nome);
        contador++;
    }
    
    document.write("Você digitou " + contador + " nome(s). ");
    document.write("Nomes informados: ");
    for (var i = 0; i < nomes.length; i++) {
        document.write(nomes[i] + "<br>");
